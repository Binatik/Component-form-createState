# Component-createElement
 
Enter the path in input.  

Creating a new value for the existing type 

You can enter different types of data 

### Types 
```sh
 {type: 'panel'}
```  

```sh
 {type: 'label'}
```  

```sh
 {type: 'button'}
``` 
 
 ### settings  
  
 ```js 
 //panel
 { 
    width: 500,
    height: 200,
    visible: false
 }  
 ``` 
 
  ```js 
 //label
 { 
    caption: 'test',
    visible: false
 }  
 ``` 
 
   ```js 
 //button
 { 
    width: 150,
    height: 100,
    visible: false
 }  
 ``` 
 
 You can change the type by the specified path.
